Команда для установки переменной среды ffmpeg:





Чтобы установить переменную среды в командной строке в Windows, используйте set:

bash
Copy code
set FFMPEG_PATH=C:\путь\к\ffmpeg
В Unix-подобных системах (Linux, macOS), используйте export:

bash
Copy code
export FFMPEG_PATH=/путь/к/ffmpeg
Обратите внимание, что эти команды устанавливают переменную среды только для текущей сессии командной строки.
Если вы хотите, чтобы эта переменная сохранялась между сессиями,
добавьте команду в ваш файл настройки оболочки (например, .bashrc или .bash_profile для Bash).
